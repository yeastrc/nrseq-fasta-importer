
This is a Java web app.

The context is normally nrseq-fasta-importer but it is not hard coded.


To Build:

To build this web app using "ant" using the ant build file "build.xml",
type "ant" at the command line in this directory.

A war file named "nrseq-fasta-importer.war" will be created and placed in the 
subdirectory "deploy".
